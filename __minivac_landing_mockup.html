A Relay Computer from 1961

Before microchips, before transistors, computers were built with mechanical relays. This is a working simulation of the Minivac 601 - watch it count in binary, the same way your phone does, just slow enough to see.

[ Full Minivac 601 Simulator Here ]
[ Running: 3-bit Binary Counter ]
[ Relays clicking, lights blinking 0-7 ]

Each relay click is a logic operation

The same operations happening billions of times per second in your phone, computer, or smartwatch. Here they're slowed down enough that you can see, hear, and understand exactly how digital logic works.

The three lights above are counting in binary: 000, 001, 010, 011, 100, 101, 110, 111 - that's 0 through 7. Every computer in the world counts this way.

Now you try: Control an elevator

This circuit uses the same relays to control a 3-floor elevator. Press buttons 1, 2, or 3 to call the elevator to different floors.

[ Minivac 601 - Elevator Controller ]
[ Buttons 1/2/3 visible and clickable ]
[ Wheel shows current floor ]

Notice how pressing a button causes relays to switch, which turns the motor wheel to the correct position. This is a real working elevator controller - the same logic used in 1960s buildings.

About this project

The Minivac 601 was created in 1961 by Claude Shannon and others as an educational computer kit. This simulator recreates the original hardware using accurate electrical simulation.
